Hosting Gratis Menggunakan Netlify.app

Terakhir diperbarui: 30 Desember 2024

    Bagikan

Netlify adalah perusahaan komputasi awan berbasis di San Francisco yang menawarkan layanan hosting dan backend tanpa server untuk aplikasi web dan situs web statis. Perusahaan ini menyediakan hosting untuk situs web yang berkas sumbernya disimpan dalam sistem pengendali versi Git yang kemudian dibuat menjadi konten web statis dan disajikan melalui Jaringan Pengiriman Konten (CDN).

Menyiapkan Git, Github, dan Netlify

Pertama-tama kita akan men-hosting website kita secara gratis menggunakan Netlify, berikut hal yang perlu dipersiapkan.

1. Install Git di Komputer

Sebelum mulai, pastikan kamu sudah menginstal Git di komputer kamu. Git diperlukan untuk versi kontrol dan mengupload project ke GitHub.

Langkah-langkah Install Git:

1. Download Git

  • Kunjungi Git Downloads
  • Pilih versi sesuai dengan sistem operasi yang kamu gunakan (Windows, macOS, atau Linux).
  • Setelah selesai mengunduh, jalankan file installer dan ikuti instruksi yang ada

Verifikasi Instalasi Git

Setelah instalasi selesai, buka terminal (Command Prompt di Windows atau Terminal di macOS/Linux), dan ketik perintah berikut: git --version

Jika Git terpasang dengan benar, maka akan muncul versi Git yang terinstal

2. Upload Project ke GitHub Menggunakan Git

Selanjutnya, kita akan meng-upload website ke GitHub. Pastikan kamu memiliki akun GitHub sebelum melanjutkan.

Langkah-langkah Upload ke GitHub:

    1. Buat Repository di GitHub:

    • Masuk ke GitHub dan login ke akun kamu atau daftar jika belum punya.
    • Setelah masuk ke akun github, lalu klik tombol New atau Create Repository pada bagian navigasi sebelah kanan.
    • Isi nama repository, misalnya: Mbagus-burhan.
    • Pilih Public agar repositori kita bisa diakses oleh siapa saja (untuk hosting gratis).
    • Jangan centang "Initialize this repository with a README" karena kita akan meng-upload file project langsung.

    2. Inisialisasi Git di Folder Project:

    • Buka folder project, klik kanan kemudian pilih open in terminal
    • Tambahkan semua file website ke staging area, gunakan perintah git add .
    • Commit file yang sudah ditambahkan, gunakan git commit -m "first commit"
    • Salin URL repository GitHub, misalnya: https://github.com/BBguss/MBagus-Burhan
    • Gunakan perintah git remote add origin (link githubnya) untuk menghubungkan folder project dengan github
    • Setelah itu push file ke github gunakan perintah git push -u origin master
    • Kemudian, periksa di GitHub, apakah file-file project sudah berhasil diupload.

    3. Daftar Akun di Netlify dan Hubungkan dengan GitHub

    • Buka Website Netlify atau https://www.netlify.com/
    • Setelah itu Klik Sign Up dan pilih untuk mendaftar menggunakan akun GitHub. Dengan memilih GitHub, Netlify akan meminta izin untuk mengakses repositorimu.
    • Setelah login, klik tombol New site from Git
    • Pilih GitHub sebagai penyedia Git
    • Netlify akan menampilkan daftar repository GitHub kita. Pilih repository yang sudah kita upload tadi
    • Terakhir pada halaman Deploy settings, pilih branch yang akan dideploy, biasanya main atau master.
